# F1P5-Standings
A Python program to grab the latest F1 results and calculate Formula 1.5 results and current standings.

## Automation
The program will be run every Monday to update the current season [data](/Data).  
(If time allows, manual runs may be done through a race weekend for quicker updates)

## Formula 1.5
For more information about Formula 1.5, please visit the [F1P5 SubReddit](https://www.reddit.com/r/Formula1Point5/)
